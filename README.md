# Võ Huy Dâng - Fullstack Developer Portfolio

This is the source code for my professional resume and portfolio website. The site highlights my experience, skills, and projects, built with a modern and responsive Bento-style layout.

## Live Deployment

The portfolio is deployed and accessible at:
[https://resume.vohuydang.io.vn/](https://resume.vohuydang.io.vn/)

## Technology Stack

* React
* TypeScript
* Tailwind CSS
* Framer Motion

## Local Development

To run this project locally:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
